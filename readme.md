# Однопоточное приложение для визуализации броуновского движения

Данное приложение визулизирует броуновское движение - хаотичное движение частиц. Частицы ограничены рамками сцены, внутри которой они находятся. Чтобы изменить размеры сцены в файле defs.h следует изменить соответсвующие макросы.

Для работы программы необходима библиотека QtWidgets.
